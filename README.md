# jasmine-nodal
Adaptation of [jasmine 2.0](http://jasmine.github.io/) for [Node.js](http://nodejs.org/).

# Usage
```
npm install jasmine-nodal
./node_modules/.bin/jasmine-nodal test-case1.js test-case2.js ...
```
